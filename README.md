# Real-Time Weather Checker

A responsive web application built using HTML, CSS, and JavaScript that provides real-time weather updates for any city worldwide. Users can search for a city by clicking the search icon to instantly view current weather conditions, including temperature, humidity, and wind speed.

## Features

- Responsive design for desktop and mobile devices
- Real-time weather data for any city
- Search functionality with clickable search icon
- Displays:
  - Temperature
  - Weather condition
  - Humidity
  - Wind speed
- Error handling for invalid or incorrect city names
- Clean and user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Weather API

## How to Use

1. Enter the city name in the search bar
2. Click the search icon
3. View live weather details instantly
4. If an invalid city is entered, an error message is displayed
